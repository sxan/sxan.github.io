Hexo+Next 搭建自己的博客

发表于 2016-06-26   |   分类于 Hexo

每次重新搭这个博客都要各种搜索，所以这一次就自己写一下流程，方便下次再用，仅此留念。

参考资料

  • Node.js官网
  • Next官网
  • Hexo官方中文文档

安装

  • 安装node.js 在官网直接下载最新版的安装包安装(必须)
  • 安装Git（必须）
  • 安装hexo
    npm install-g hexo或 npm install hexo-cli -g(建议)

初始化hexo

  • 创建一个文件夹 例如:blog
  • 然后在该文件夹下执行命令(shift+右键 在此处打开命令窗口)
    hexo init
  • 简单配置
    打开blog/_config.yml文件建议修改以下字段(注意空格)

测试
hexo server默认4000端口 或hexo server -p 9000我电脑4000被占用所以改了端口
网页打开http://localhost:4000或http://localhost:9000

更换Next主题

  • 下载主题(同样是在刚才的目录下执行)
    git clone https://github.com/iissnan/hexo-theme-next themes/next或直接下载压缩包解压到themes目录下
  • 更换主题
    将blog/_config.yml的theme: landscape改为theme: next 注意空格
  • 修改配置
    默认是Muse 我用的是Mist三种主题 可以自己都试试 #号是注释的

现在可以重启一下服务器然后重新访问或者直接重新访问
